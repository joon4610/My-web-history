HTML 이란 무엇인가?

Markup Language(HTML) is the standard markup Language for creating web pagesand web applications.

현재까지 배운 tag

  • h1 ~ h6(제목태그)
  • strong(굵게), u(밑줄)
  • br(줄바꿈), p(단락)
  • img(사진)
  • li, ul(li태그를 그룹핑), ol(앞에 숫자 붙여서 그룹핑)
  • title(제목), meta charset(~~로 글자를 읽어라)
  • head(본문에 대한 설명을하는 태그), body(본문)
  • html(head, body를 다 감싸는 태그), DOCTYPE(이 문서는 ~~다)
  • a(anchor=링크)
    Attribute: href(참조=주소), target=_blank(새창으로 열기), title(설명)
  • style(여기는 CSS문법으로 해석해라)
  • class(같은 그룹으로 묶어주는 tag), id(class와 동일한 기능이지만 더 우선적)
    class 는 .으로 id는 #을 앞에 붙이는 것으로 선택
    tag선택자로 전체적인 디자인을 하고 class, id(=웹페이지에 하나만 사용하는 것을 권장하는 tag)로 예외상황을 묶어서 디자인하기 위함
  • div, span(디자인을 하기 위한 무색무취의 의미 없는 태크, div is block tag, span is lnline tag)
