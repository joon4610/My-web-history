What is CSS?

디자인하는 HTML과는 다른 새로운 언어
따라서 HTML에게 CSS를 인식하게 하는 언어가 필요
그것이 style tag (=HTML아 여기 안에있는 언어는 CSS문법으로 해석해라~ 라는 뜻)

    웹페이지에 CSS를 삽입하는 방법

  1. style tag
  2. style 속성
CSS를 HTML tag에 (style="") 형식으로 추가할 수 도 있음
+ style="" 은 HTML의 속성

빨간색은 selector(선택자), 노란색은 declaration(효과, 선언)
selector 는 효과를 어디에 줄지 "선택"
declaration은 효과의 "내용"
;은 구분자로 사용
여기서 color 는 property(속성), black은 value(값)

+ 코딩을 잘하는 법 중 하나는 중복의 제거
1억개의 a tag의 색을 바꿔야한다면? - 일괄적으로 바꿀 수 있는 방법이 좋은 방법임
+ HTML에 문법으로 디자인을 추가하면 정보로서의 가치가 줄어듦
ex) 검색시 디자인에 관한것을 무시하고 HTML만을 검색하게 만들 수 있게 하는것이 CSS
